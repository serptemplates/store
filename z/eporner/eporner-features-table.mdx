# Eporner Video Downloader - Supported Video Types

| Video Type                 | Platform           | Download Method     | Status   | Notes                             |
| -------------------------- | ------------------ | ------------------- | -------- | --------------------------------- |
| **HD Porn Videos**         | eporner.com        | yt-dlp extraction   | ✅ Works | Main HD video pages               |
| **Standard Videos**        | eporner.com        | yt-dlp extraction   | ✅ Works | Regular video pages               |
| **Embed Videos**           | eporner.com        | yt-dlp extraction   | ✅ Works | Embedded video players            |
| **WWW Videos**             | www.eporner.com    | yt-dlp extraction   | ✅ Works | WWW subdomain support             |

## Supported Formats

- **MP4** (Primary format for Eporner videos)
- **Various quality levels** (HD and standard resolutions)
- **Thumbnail extraction** (from og:image meta tags)
- **Metadata extraction** (title, duration, description, views)

## URL Patterns Supported

- `https://eporner.com/hd-porn/[video-id]/[title]/` (HD porn videos)
- `https://eporner.com/video-[video-id]/[title]/` (Standard videos)  
- `https://eporner.com/embed/[video-id]` (Embedded videos)
- `https://www.eporner.com/*` (WWW subdomain)

## Requirements

- Chrome/Chromium browser only
- Valid license key required (uses store.externulls.com)
- JavaScript enabled
- One video at a time (no batch downloads)
- Internet connection for extraction

## Extraction Methods

| Method                     | Priority | Description                           |
| -------------------------- | -------- | ------------------------------------- |
| **yt-dlp Extractor**       | Primary  | Uses yt-dlp extraction logic          |
| **Page Metadata**          | Primary  | Extracts from HTML meta tags         |
| **Inject Script**          | Support  | Page data injection for video IDs    |
| **URL Pattern Matching**   | Support  | Multiple URL pattern recognition      |

## Technical Features

- **Service Worker Architecture** (Manifest V3)
- **Content Script Injection** (for page analysis)
- **Context Menu Integration** (right-click downloads)
- **Offscreen Document Support** (background processing)
- **Storage Management** (license and settings)
- **Cross-domain Support** (multiple Eporner domains)

## Limitations

- **Single Platform**: Only works with Eporner.com
- **No Bulk Downloads**: One video at a time
- **License Required**: Needs valid activation
- **Chrome Only**: No Firefox/Safari support
- **No Live Streams**: Static video content only
- **No Playlists**: Individual videos only
