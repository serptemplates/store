# Supported Sources

| Feature | Location | Notes |
| --- | --- | --- |
| **YouTube Watch Pages** | youtube.com/watch, Shorts, live replays | Progressive + adaptive formats with FastStream merge |
| **Embedded Players** | iframe embeds on third-party sites | Falls back to embed detection when not on youtube.com |
| **Download Manager** | In-page overlay + popup | Shared queue UI with cancel/clear actions |
| **yt-dlp Helper** | Popup quick-copy buttons | Copies platform-specific commands for power users |
