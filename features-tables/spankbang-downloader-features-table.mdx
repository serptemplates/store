# SpankBang Video Downloader - Supported Video Types

| Video Type             | Platform                | Download Method        | Status     | Notes                              |
| ---------------------- | ----------------------- | ---------------------- | ---------- | ---------------------------------- |
| **Single Videos**      | SpankBang video pages   | Stream URL extraction  | ✅ Works   | Individual video downloads only    |
| **Play Pages**         | Alternative video URLs  | Stream key API         | ✅ Works   | Different URL format support       |
| **Embed Videos**       | SpankBang embeds        | Embed page parsing     | ✅ Works   | Embedded video players             |
| **Mobile Videos**      | Mobile SpankBang        | Mobile page detection  | ✅ Works   | m.spankbang.com support            |
| **Multiple Qualities** | All SpankBang videos    | Format preference      | ✅ Works   | 240p, 480p, 720p, 1080p available |
| **Context Menu**       | Right-click download    | Background integration | ✅ Works   | Download via right-click option    |
| **Progress Tracking**  | All downloads           | Chrome Downloads API   | ✅ Works   | Real-time progress with file size  |

## Supported Formats

- **MP4** (Primary and only supported format)
- **Multiple resolutions** (240p, 480p, 720p, 1080p when available)
- **Automatic quality sorting** (Highest quality selected by default)
- **Format filtering** (Skips HLS/m3u8 and DASH/mpd formats)

## Requirements

- Chrome/Chromium browser only (Manifest V3)
- Valid license key required ($9 product purchase)
- Internet connection for license validation
- Must be on SpankBang domain pages

## Supported SpankBang URLs

- Video pages: `https://spankbang.com/{id}/video/{title}`
- Play pages: `https://spankbang.com/{id}/play/{title}`
- Embed pages: `https://spankbang.com/{id}/embed`
- Mobile site: `https://m.spankbang.com/*`
- All SpankBang subdomains: `https://*.spankbang.com/*`

## User Interface

- **License activation** - Email and license key validation system
- **Video metadata display** - Thumbnail, title, uploader, duration, views, likes
- **Quality selector** - Dropdown with available MP4 formats
- **Download progress** - Real-time progress bar with percentage and file size
- **Context menu integration** - Right-click download option
- **Help system** - Built-in usage instructions
- **Error handling** - Clear messages for various failure scenarios

## Technical Features

- **Stream URL extraction** - Parses page script variables for video URLs
- **Stream key API** - Fetches from `/api/videos/stream` endpoint
- **Legacy initials fallback** - Uses `window.initials.videoModel` data
- **XPlayer settings** - Processes `window.initials.xplayerSettings` sources
- **Dynamic content monitoring** - Mutation observer for video player detection
- **Header-based fetching** - Content script downloads with proper referer
- **Blob URL creation** - Secure download handling

## Important Limitations

- **Single video downloads** - No batch or playlist support
- **SpankBang exclusive** - Only works on SpankBang domains
- **License required** - All functionality blocked without $9 activation
- **MP4 format only** - No support for HLS/m3u8 or DASH/mpd formats
- **Chrome only** - No support for other browsers
- **No format selection** - Auto-selects highest quality available
- **No download history** - No record of previous downloads
- **Online dependency** - Requires internet for license validation
