## Supported Formats

- **MP4** (Primary format - direct downloads)
- **HLS/M3U8** (Streaming format with segment processing)
- **Multiple resolutions** (240p, 360p, 480p, 720p, 1080p when available)
- **Quality auto-detection** (Parsed from URL patterns like _720p.mp4)
- **HLS segment merging** (Automatic via offscreen worker)
- **Bitrate extraction** (From URL parameters when available)

## Technical Features

- **Multi-source extraction** - Scans scripts, video elements, and meta tags
- **Candidate URL collection** - Finds all playable video URLs on page
- **Pattern matching** - Detects URLs from JavaScript variables (video_url, videoUrl, file, src)
- **DOM extraction** - Video metadata from meta tags and page elements
- **Content script injection** - Automatic video detection
- **Declarative Net Request** - CORS header modification for CDN access
- **Background processing** - Download management via service worker
- **Offscreen document** - HLS segment processing and MP4 downloads
- **HLS segment merger** - Combines .ts segments into single MP4 file
- **Progress restoration** - Resumes UI state after page refresh
- **Cross-context communication** - Page context to content script to background
- **Format normalization** - Quality parsing from URL patterns (_480p.mp4, /720p/, etc.)
- **Referer injection** - Dynamic header modification for ahcdn.com requests

## Important Limitations

- **Single video downloads** - No batch or playlist support
- **AlphaPorno network only** - Limited to AlphaPorno domains
- **License required** - All functionality blocked without activation
- **Adult content only** - 18+ age restriction applies
- **Chrome only** - No support for other browsers
- **No download history** - No persistent storage of past downloads
- **Internet dependency** - Requires connection for license validation and video access
