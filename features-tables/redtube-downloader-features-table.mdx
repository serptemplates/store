# RedTube Video Downloader - Supported Video Types

| Video Type             | Platform                | Download Method        | Status     | Notes                              |
| ---------------------- | ----------------------- | ---------------------- | ---------- | ---------------------------------- |
| **Single Videos**      | RedTube video pages     | yt-dlp extraction      | ✅ Works   | Individual video downloads only    |
| **Embed Videos**       | RedTube embeds          | Embed page parsing     | ✅ Works   | Supports embed.redtube.com URLs    |
| **HLS Streams**        | RedTube CDN delivery    | M3U8 playlist parsing  | ✅ Works   | Segments downloaded and merged     |
| **Multiple Qualities** | All RedTube videos      | Format preference      | ✅ Works   | 240p, 480p, 720p+ available       |
| **Context Menu**       | Right-click download    | Background integration | ✅ Works   | Download via right-click option    |
| **Progress Tracking**  | All downloads           | Real-time monitoring   | ✅ Works   | Visual progress bar with bytes     |

## Supported Formats

- **MP4** (Primary format)
- **HLS/M3U8** (Processed into segments then concatenated)
- **Multiple resolutions** (240p, 480p, 720p, 1080p when available)
- **Quality auto-detection** (Based on format_id patterns)
- **Format deduplication** (Removes duplicate URLs)

## Requirements

- Chrome/Chromium browser only (Manifest V3)
- Valid license key required (Gumroad verification)
- Internet connection for API calls
- Age verification (18+ content)

## Supported RedTube URLs

- Video pages: `https://www.redtube.com/[video-id]`
- Brazilian domain: `https://www.redtube.com.br/[video-id]`
- Embed pages: `https://embed.redtube.com/?id=[video-id]`
- All RedTube network domains

## User Interface

- **License activation** - Email and license key validation system
- **Video metadata display** - Thumbnail, title, duration, views, uploader
- **Quality selector** - Dropdown with user-friendly quality labels
- **Download progress** - Real-time progress bar with speed and file size
- **Error handling** - Clear error messages for various failure scenarios
- **Context menu integration** - Right-click download option
- **Help system** - Built-in help text for user guidance

## Technical Features

- **yt-dlp adaptation** - JavaScript implementation of Python extractor
- **Multiple extraction sources** - Page scripts, media definitions, source tags
- **HLS processing** - Downloads and concatenates video segments
- **FFmpeg integration** - Offscreen processing for complex operations
- **IndexedDB storage** - Temporary storage for large video segments
- **Cross-domain handling** - Proper headers and CORS management
- **Dynamic detection** - Monitors page changes for new videos

## Important Limitations

- **Single video downloads** - No batch or playlist support
- **RedTube exclusive** - Only works on RedTube domains
- **License required** - All functionality blocked without activation
- **Adult content only** - 18+ age restriction applies
- **Chrome only** - No support for other browsers
- **No live streams** - On-demand video content only
- **Local downloads only** - No cloud storage integration
- **No download history** - No record of previous downloads
