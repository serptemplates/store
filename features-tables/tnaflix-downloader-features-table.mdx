# TNAFlix Video Downloader - Supported Video Types

| Video Type             | Platform                | Download Method        | Status     | Notes                              |
| ---------------------- | ----------------------- | ---------------------- | ---------- | ---------------------------------- |
| **Single Videos**      | TNAFlix video pages     | yt-dlp extraction      | ✅ Works   | Individual video downloads only    |
| **Multiple Qualities** | All TNAFlix videos      | Format preference      | ✅ Works   | Multiple resolutions available     |
| **Progress Tracking**  | All downloads           | Chrome Downloads API   | ✅ Works   | Real-time progress with file size  |

## Supported Formats

- **Format support varies** (Detected from video sources)
- **Multiple resolutions** (Quality depends on video availability)
- **Automatic quality detection** (Highest quality selected by default)

## Requirements

- Chrome/Chromium browser only (Manifest V3)
- Valid license key required ($9 product purchase)
- Internet connection for license validation
- Must be on TNAFlix domain pages

## Supported TNAFlix URLs

- TNAFlix pages: `https://tnaflix.com/*`
- TNAFlix pages: `https://www.tnaflix.com/*`
- API access: `https://store.externulls.com/*`

## User Interface

- **License activation** - Email and license key validation system ($9)
- **Video metadata display** - Thumbnail, title, duration, likes/dislikes
- **Quality selector** - Dropdown with available formats
- **Download progress** - Real-time progress bar with percentage
- **Help system** - Built-in usage instructions
- **Error handling** - Clear messages for various failure scenarios

## Technical Features

- **yt-dlp extraction** - Uses yt-dlp-based video detection
- **Content script injection** - Automatic video detection on TNAFlix pages
- **Background service worker** - Download management and processing
- **Offscreen processing** - Advanced video processing capabilities
- **Context menu integration** - Right-click download options

## Important Limitations

- **Single video downloads** - No batch or playlist support
- **TNAFlix exclusive** - Only works on TNAFlix domains
- **License required** - All functionality blocked without $9 activation
- **Chrome only** - No support for other browsers
- **Internet dependency** - Requires connection for license validation
- **No download history** - No record of previous downloads
